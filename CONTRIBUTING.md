# Contributing

Thanks for contributing to **dc-manager**!

## Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the dev server:
   ```bash
   npm run dev
   ```

## Scripts

- `npm run lint` / `npm run lint:fix`
- `npm run format` / `npm run format:fix`
- `npm run typecheck`
- `npm run test`
- `npm run e2e`

## Pull Requests

- Keep changes focused and include tests when applicable.
- Ensure `npm run ci` passes before submitting.
- Document user-facing changes in README and docs where appropriate.
